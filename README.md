## Setup Instructions

1. **Clone the repository**

   ```sh
   git clone git@github.com:dev-rathore/reactjs-frontend.git
   cd reactjs-frontend
   ```

2. **Install Dependencies**

   ```sh
   npm install
   ```

3. **Set Up Environment Variables**

   **Copy `.env.example` to `.env`**

   ```sh
   cp .env.example .env
   ```

4. **Start the frontend**

   ```sh
   npm run dev
   ```
